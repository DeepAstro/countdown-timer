# Assets Directory

This directory contains application assets.

## Sounds

Place notification sound files here:
- `notification.wav` - Default notification sound
- `notification.mp3` - Alternative MP3 format
- `alarm.wav` - Alarm sound

## Icons

Place application icons here:
- `app_icon.ico` - Windows application icon (256x256 recommended)

## Note

If no sound files are present, the application will use the system beep sound.
